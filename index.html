Curriculum Vitae

El primer proyecto que hice fue crear un CV en donde se incluyeran listas y links, utilizando solo el HTML el cuál recién se nos había enseñado.

Landing Page

En este proyecto tuvimos que crear una "landing page", utilizando el css que se nos había enseñado y tablas.

Iguana Page

Acá se nos proporciono una maqueta por Adobe XD de una landing page, la cuál tuvimos que replicar con el contenido flexbox que se nos enseño. Al ser una maqueta, contamos solo con la información de los colores o tamaños de letras.

Cuppon

El cuarto proyecto se trató de crear una página web para vender productos utilizando Bootstrap, contando con una vista tanto para escritorio, como para teléfonos.

Github

En este tuvimos que contruir nuevamente un CV, pero a partir de una plantilla de un repositorio de Github, por lo que aprendimos a hacer fork y clone.
